# consult-git-ls.el

Go to files using `git ls-files` via [Consult](https://github.com/minad/consult).

## Requirements

* GNU Emacs >= 27.1
* Consult >= 1.6
* Git

## Usage

### `consult-git-ls`

You can call this interactively, or bind to your keybind.
